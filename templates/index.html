Crop Recommendation System Crop Yield Prediction

Project Overview

This system is developed as part of the Data Visualization Laboratory.

It leverages Machine Learning models to provide actionable agricultural predictions.

Crop Recommendation System

Uses Random Forest Classifier trained on soil & climate parameters to suggest the most suitable crop.

Crop Yield Prediction

Uses Random Forest Regressor to estimate the potential yield based on environmental conditions.

Dataset Information

Dataset Description Model Used
crop_recommendation.csv Contains soil nutrients (N, P, K), temperature, humidity, pH, rainfall, and crop labels for training the recommendation system. Random Forest Classifier
Crop_Yield.csv Contains crop-specific yield data along with climatic and soil conditions used for regression analysis. Random Forest Regressor
